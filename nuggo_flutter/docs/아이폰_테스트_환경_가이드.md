## 1단계: 개발자 모드 활성화 (필수)

### 방법 A: 설정에서 직접 켜기

1. 아이폰에서 **설정** 앱 실행
2. **개인정보 보호 및 보안** 탭
3. **아래로 스크롤** → **개발자 모드** 선택
4. 토글을 **켬**으로 변경
5. **재부팅** 후, "개발자 모드를 켜시겠습니까?" → **켜기** 선택

### 방법 B: 개발자 모드 메뉴가 안 보일 때

일부 iOS 버전에서는 **처음 연결 시에만** 개발자 모드 메뉴가 나타납니다.

1. **Xcode** 실행 (맥)
2. 아이폰을 **USB로 맥에 연결**
3. 상단 메뉴: **Window** → **Devices and Simulators** (⇧⌘2)
4. 왼쪽에서 **YS의 iPhone** 선택
5. 아이폰 화면에서 **잠금 해제** 유지
6. Xcode에서 **+** 버튼으로 새 앱 추가 후 실행하거나, 기존 프로젝트 열어서 실행 시도

→ 이 과정 후 **설정 > 개인정보 보호 및 보안**에 **개발자 모드** 항목이 나타날 수 있습니다.

### 방법 C: Xcode로 Runner 프로젝트 직접 실행

1. Finder에서 아래 경로로 이동:
   ```
   /Users/yongsangjo/커서/nuggo/nuggo_flutter/ios/
   ```
2. **Runner.xcworkspace** 더블클릭 (`.xcodeproj`가 아님)
3. 상단 디바이스 선택에서 **YS의 iPhone** 선택
4. **▶ Run** 버튼 클릭
5. 처음 실행 시 **Signing** 설정 필요:
   - **Runner** 프로젝트 선택 → **Signing & Capabilities** 탭
   - **Automatically manage signing** 체크
   - **Team**에서 본인 Apple ID 선택 (없으면 Add Account로 추가)

## 2단계: Flutter로 앱 실행

개발자 모드 활성화 후, 터미널에서:

```bash
cd /Users/yongsangjo/커서/nuggo/nuggo_flutter
flutter run -d "YS의 iPhone"
```

또는 기기 ID로:

```bash
flutter run -d 00008130-000918EA14A2001C
```

## 문제 해결

| 증상 | 해결 |
|------|------|
| "enable Developer Mode" 메시지 | 위 1단계 개발자 모드 활성화 |
| "No development team" | Xcode → Signing & Capabilities → Team 선택 |
| "Trust this developer" | 아이폰 설정 → 일반 → 기기 관리 → 개발자 앱 → 신뢰 |
| `flutter devices`에 아이폰 안 보임 | 케이블 재연결, 잠금 해제, 맥/아이폰 재부팅 |
